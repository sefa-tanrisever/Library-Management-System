Library Management System
This project is a robust Library Management System developed to automate inventory tracking, member records, and borrowing transactions. It demonstrates the integration of Python backend logic with SQL database management.

Key Features
Inventory Management: Efficiently add, update, and remove book records (ISBN, Title, Author, Category).

Member Tracking: Manage library memberships and personal data securely.

Borrowing System: Automated system for tracking borrowed books, return dates, and availability status.

Database Integration: Utilizes structured queries for data persistence and integrity.

Tech Stack
Language: Python

Database: SQL (PostgreSQL / MySQL)

Concepts: CRUD Operations, Relational Database Design, Backend Development.
